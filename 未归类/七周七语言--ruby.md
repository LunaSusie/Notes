# ruby命令行
* ruby是解释执行。
* 不用声明变量。
* 每一条代码都有返回值。
# ruby的编程模型
* 纯面向对象语言。一切都是对象，易于获取对象的任何信息。
* ryby强类型语言。代码执行时候进行类型检查，叫动态类型。
* 定义函数，数组，散列表。
* 代码块和yield。
* 从文件运行ruby。
* 定义类。一切类继承object。
* 面向对象语言利用继承，将行为传播到相似的对象上。若对象想继承并不相似的多种行为，一方面允许多继承，另一方面通过其他方案。ruby通过模块实现，模块是函数和常量的集合，一个类包含了一个模块，这个模块的行为和常量也就成为了类的一部分。
* mixin。
